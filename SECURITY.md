# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

OSINT FOX is an open-source intelligence tool for authorized security research only.

If you discover a security vulnerability, please do NOT file a public issue. Instead, send a private description to the repository owner via GitHub.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We aim to respond to vulnerability reports within 48 hours.
